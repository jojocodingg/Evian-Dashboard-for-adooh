# Evian adOOH Post-Campaign Analysis Dashboard

## Overview
An interactive Power BI dashboard analysing a 15-day 
outdoor advertising campaign for Evian Natural Mineral 
Water across the Paris bus network (January 2017).

## Key Results
- 374,090 total impressions across 71 locations
- 91.7% ad completion rate vs 75% industry benchmark
- +26.18% campaign momentum Week 2 vs Week 1
- 73.87s average dwell time against 15s ad slot

## Dashboard Pages
1. Executive Overview — campaign KPIs and trends
2. Customer Analysis — audience type and engagement
3. Predictive Model — impression forecasting
4. Geographic Analysis — location-level performance map

## Tools Used
- Power BI — dashboard and visualisation
- R — time series forecasting (ETS model)
- Python — data analysis and preprocessing

## Data
- `EVIAN_res4.csv` — audience and impression data (primary dataset)

Note: Playback data (110,400 ad plays across 15 days) was used 
for the Impressions Per Play KPI but is excluded from this 
repository due to file size constraints.

## How to Use
1. Download the .pbix file
2. Open in Power BI Desktop
3. Data will load automatically from the CSV files
